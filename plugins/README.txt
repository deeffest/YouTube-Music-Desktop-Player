INSTRUCTIONS FOR CUSTOM PLUGINS:

1. OPTION A (Application Folder):
   Drag the .js scripts you want to enable as custom plugins 
   into the "plugins" folder located in the root directory of application.

2. OPTION B (Home Folder - Recommended for Linux):
   Go to your home directory:
   - Windows: C:\Users\USERNAME\
   - Linux:   /home/USERNAME/

   Step 1: Enable "Show hidden files and folders" in your file manager.
   Step 2: Find the "Youtube-Music-Desktop-Player" folder.
   Step 3: Inside it, create a folder named "plugins".
   Step 4: Add your .js scripts there.

Note: Plugins from the home folder are useful if the application 
is installed in a restricted system directory like /opt/ or 
C:\Program Files\.
